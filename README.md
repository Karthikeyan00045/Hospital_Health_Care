# Hospital Health Care Performance Dashboard

## 📖 Overview
This project demonstrates end-to-end data analysis and visualization using SQL, Python, Excel, and Tableau.  
The goal is to transform raw hospital patient records into actionable insights for healthcare performance.

## 🛠️ Tech Stack
- SQL: Database queries and schema design
- Python: Data cleaning, preprocessing, and automation
- Excel: Structured datasets and quick analysis
- Tableau: Interactive dashboard for storytelling

## 📊 Dashboard Highlights
- Patients by State
- Geographic Spread
- Gender Distribution
- Age Distribution
- KPIs at the Top

## 🚀 Outcomes
- Cleaned and normalized hospital dataset
- Automated workflows in Python
- Recruiter-ready Tableau dashboard
- Transparent documentation for reproducibility

## 📂 Repository Structure
- `/data` → Raw and cleaned datasets
- `/sql` → Database scripts
- `/python` → Cleaning and analysis scripts
- `/tableau` → Dashboard files
